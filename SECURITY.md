# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in **WiFi-PWN Toolkit**, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please report via:
- **Email:** jojinjohn1709@gmail.com
- **LinkedIn:** [jojin-john](https://www.linkedin.com/in/jojin-john/)

Please include:
1. A description of the vulnerability
2. Steps to reproduce
3. Potential impact
4. Suggested fix (if any)

I will acknowledge receipt within **48 hours** and provide a detailed response within **7 days** indicating the next steps.

Thank you for helping keep this project secure.
